# Cancer Cell Prediction (XPRO Project)

A Machine Learning-based cancer cell classification system developed as part of the XPRO program by Employability.life.  
The model analyzes medical diagnostic features and predicts whether a tumor is **Malignant** or **Benign**.

## Project Overview

Early detection of cancer significantly improves survival rates.  
This project applies supervised machine learning techniques to classify tumor cells based on extracted medical features.

The system includes:

- Data preprocessing
- Model training
- Performance evaluation
- Prediction system

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Approach

- Synthetic patient data generation
- Adding anomalies to mimic real-world medical data
- Data Cleaning & Preprocessing
- Feature Selection
- Train-Test Split
- Model Training using Logistic Regression, Decision Tree, Random Forest, SVM, KNN, Gradient Boosting
- Model Evaluation using:
  - Accuracy Score
  - Confusion Matrix
  - ROC curve
  - Classification Report
